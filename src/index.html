Convert Wikipedia Tables to CSV

Enter the URL of the Wikipedia page containing the table(s). Press "Convert" and copy the result to your clipboard.

Input

e.g. https://en.wikipedia.org/wiki/Lists_of_earthquakes
Options
Default is "table.wikitable"

Output (clear)

No output generated.

Tip: to use the data in Excel or similar spreadsheet applications paste the result from your clipboard into the first cell of your spreadsheet. Set the delimiter character to "comma".
